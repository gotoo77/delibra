# Minimal Treasure Hunt Design Input

theme: Cabinet de curiosites disparu dans une bibliotheque municipale
target_players: 4 adultes, joueurs occasionnels, aiment les enigmes mais pas les codes trop abstraits
duration: 60 minutes
difficulty: intermediaire
domain_constraints: histoire locale fictive, botanique simple, observation visuelle, logique de classement
physical_constraints: jouable en interieur, sans internet, avec papier, enveloppes, livres factices, cadenas a 3 chiffres
desired_tone: mysterieux, chaleureux, legerement humoristique, sans horreur
