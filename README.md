# Delibra

Delibra orchestrates artifact derivation. Models reason; Delibra preserves the resulting artifacts and trace.

## Local Development

Install the CLI in editable mode:

```bash
python3 -m pip install -e .
```

Run the CLI:

```bash
delibra --help
delibra validate --help
delibra run --help
```

Run tests:

```bash
python3 -m unittest
```
